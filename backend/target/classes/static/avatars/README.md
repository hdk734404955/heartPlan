## 文件规格

- 格式：PNG（推荐）或JPG
- 尺寸：建议 200x200 像素或更高
- 大小：每个文件不超过 500KB
- 内容：适合作为用户头像的图片，如卡通头像、抽象图案等

## 访问路径

这些头像文件将通过以下URL访问：
`http://localhost:8080/api/files/avatars/avatar_1.png`

## 注意事项

- 这些文件是系统内置的，不会被用户操作删除
- 用户选择随机头像时，系统会从这些文件中随机选择一个
- 如果文件不存在，可能会导致头像显示异常
